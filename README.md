# DICIONARIOS

Este repositório é uma coleção centralizada e organizada de listas de palavras em texto puro (`.txt`), criadas especificamente para o desenvolvimento ou teste de aplicações e games.

## Estrutura do Repositório

Os dicionários estão organizados por idioma dentro de suas respectivas pastas. Cada pasta contém o arquivo de texto com as palavras e a sua licença original.

## Créditos e Fontes Originais

Todos os dicionários presentes neste repositório são derivados de repositórios de terceiros. A tabela abaixo relaciona a pasta do idioma com a sua fonte original:

| Idioma (Pasta) | Fonte Original |
| :--- | :--- |
| Inglês (`en`) | [dwyl/english-words](https://github.com/dwyl/english-words) |
| Espanhol (`es`) | [xavier-hernandez/spanish-wordlist](https://github.com/xavier-hernandez/spanish-wordlist) |
| Francês (`fr`) | [Taknok/French-Wordlist](https://github.com/Taknok/French-Wordlist) |
| Italiano (`it`) | [napolux/paroleitaliane](https://github.com/napolux/paroleitaliane/tree/main) |
| Português (`pt-BR`) | [herzog0/palavras](https://github.com/herzog0/palavras) |
| Sueco (`sv`) | [nilsj2/swedish-words](https://github.com/nilsj2/swedish-words) |

**Aviso Importante sobre os Dicionários:**
Os arquivos contendo as listas de palavras retêm estritamente as licenças definidas por seus autores originais. Consulte o arquivo `LICENSE` >> `LICENSE.txt` localizado dentro da subpasta de cada idioma.
